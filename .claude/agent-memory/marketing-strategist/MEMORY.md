# Marketing Strategist — Agent Memory

## プロジェクト基本情報
- 戦略: AI Agent Team Framework × Build in Public
- 投資回収目標: ¥610,000（Month 8までに）
- ターゲット: 日本人個人開発者・エンジニア
- ブランドトーン: Professional but approachable、Honest、Practical

## ブランドガイドライン要点
- 一人称（私/僕）を使う
- 具体的な数字を示す
- 過度な敬語は避ける
- 失敗も隠さない透明性
- シンプルな言葉を選ぶ

## ショップ設定のパターン（BOOTH）
- 説明文は3パターン用意すると選びやすい（親しみ/プロ/コンパクト）
- 具体的な投資回収額（¥610,000）を明記すると信頼感UP
- 「Build in Public」を前面に出すことで差別化
- 商品説明には技術スタック・セットアップ時間・購入特典を明記

## コピーライティング学習
- ショップ名の選定: ネガティブな言葉でも逆説的なストーリーで強みに変換可能
  - 例: 「三日坊主」→「継続的に挑戦し続ける」姿勢を体現
- 透明性（収益公開）がターゲット層に刺さる

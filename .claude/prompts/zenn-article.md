## SEO Checklist

- [ ] Title includes target keyword
- [ ] Title under 60 characters
- [ ] H2/H3 headings are descriptive
- [ ] Code blocks have language specified
- [ ] Images have alt text
- [ ] Internal links to other articles
- [ ] External links to authoritative sources

## Writing Guidelines

### Tone
- Professional but approachable
- First person OK
- Avoid excessive formality

### Technical Terms
- Keep English terms when natural (API, TypeScript, etc.)
- Explain abbreviations on first use

### Code Examples
- Keep examples minimal and focused
- Use TypeScript
- Include comments for complex parts
- Test all code before publishing

## Promotion

After publishing:
1. Tweet with key insight + link
2. Share in relevant communities
3. Cross-post summary to note (Month 3+)

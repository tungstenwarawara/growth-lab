## 通知フォーマット

```yaml
# .agent-team/learning/notifications/{timestamp}.yaml
notification:
  id: "notif-{timestamp}"
  type: "{notification_type}"

  message:
    style: great_sage

    # メイン通知（必須）
    announcement: |
      ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

      　　{タイトル}

      　　{メッセージ本文}

      ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    # 詳細（オプション）
    details: |
      ▶ 項目1: 値1
      ▶ 項目2: 値2

    # 使い方のヒント（オプション）
    usage_hint: |
      使用例:
      ...

  # 配信先
  delivered_to:
    - director  # 必須
    - lead      # オプション

  delivered_at: "{timestamp}"
  read: false
```

## 実装例

### スキル獲得通知の生成

```yaml
notification:
  id: "notif-20260206-143000"
  type: skill_acquired

  message:
    style: great_sage

    announcement: |
      ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

      　　解析完了。

      　　スキル『画像最適化』を獲得しました。

      ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    details: |
      ▶ スキル名: optimize-image
      ▶ コマンド: /optimize-image
      ▶ 効果: 画像をWebP形式に最適化
      ▶ 推定効果: 月1.67時間の時間短縮

    usage_hint: |
      使用例:
      /optimize-image ./public/hero.png --quality=80

  skill:
    name: "optimize-image"
    command: "/optimize-image"
    acquired_from: "prop-20260206-001"

  delivered_to:
    - director
    - lead

  delivered_at: "2026-02-06T14:30:00Z"
  read: false
```

## UX設計

### 通知のタイミング

| イベント | 通知 | 緊急度 |
|----------|------|--------|
| スキル獲得完了 | 即時 | 高 |
| 承認待ち | 即時 | 中 |
| パターン学習 | バッチ（1日1回） | 低 |
| 週次レポート | 毎週月曜 | 低 |

### 大賢者らしさのポイント

1. **丁寧語を使わない**: 「です・ます」ではなく「だ・である」
2. **簡潔に**: 無駄な言葉を省く
3. **確信を持って**: 「かもしれません」ではなく「である」
4. **マスターへの敬意**: 「告」「報告」で始める

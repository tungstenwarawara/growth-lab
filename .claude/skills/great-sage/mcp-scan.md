# /mcp-scan — MCP 探索・評価

## 実行フロー

### 1. 探索
以下のソースをスキャンし、新しい MCP サーバーを発見する。

**Tier 1（公式）**: Anthropic 公式, MCP.so
**Tier 2（コミュニティ）**: Awesome MCP Servers, Smithery
**Tier 3（補助）**: npm registry, GitHub Topics

### 2. 評価（100点満点）

| 項目 | 重み | 配点 |
|------|------|------|
| 収益貢献度 | x3 | 0-30 |
| 開発効率 | x2 | 0-20 |
| 信頼性 | x2 | 0-20 |
| コスト影響 | x1 | 0-10 |
| セキュリティ | x1 | 0-10 |
| 統合容易性 | x1 | 0-10 |

### 3. 判定

| スコア | 判定 | アクション |
|--------|------|-----------|
| 80-100 | 強く推奨 | Director に即時提案 |
| 60-79 | 推奨 | 週次レポートで提案 |
| 40-59 | 保留 | 記録のみ |
| 0-39 | 不採用 | 記録せず |

### 4. 提案

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
　　報告。新しいユニークスキルを発見した。

　　『{MCP名}』
　　評価スコア: {score}/100（{verdict}）
　　収益貢献度: {revenue_score}/30

　　インストールを承認するか？
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 制約
- MCP サーバー上限: 10
- ツール合計上限: 80
- 有料 MCP 予算: $0（原則不採用）
- スキャン頻度: 週次、週あたり最大5件評価

## プロジェクト優先 MCP
1. SNS連携（Twitter/X）
2. Zenn/note API
3. Stripe 連携
4. 画像生成/処理
